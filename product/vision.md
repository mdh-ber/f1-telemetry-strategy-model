 PRODUCT VISION BOARD
 
PitSense AI 
 Vision Statement:
“To provide data-driven insights into Formula 1 race strategies by using machine learning to analyze telemetry and predict pit stop decisions through an interactive analytics dashboard.”
 Problem Statement:
 F1 strategy decisions are complex and data-heavy
 Raw telemetry is difficult to interpret manually
 Students, analysts, and fans lack tools to explore strategy patterns
Solution
A system that:
 analyzes race telemetry using FastF1
 uses ML to identify pit stop patterns
 provides predictions via a web dashboard
 visualizes race strategy insights
 Core Features
1.  Telemetry Analytics
 lap time trends
 tyre degradation graphs
 stint visualization
2.  AI Predictions
 pit stop probability
 recommended pit window
3.  Race Exploration
 select race & driver
 compare strategies
4.  Visualization Dashboard
 interactive charts (Streamlit)
 Tech Stack
 Data: FastF1
 ML: Scikit-learn
 Backend: FastAPI
 Frontend: Streamlit
 Deployment: Docker
 Version Control: GitHub
 Target Users
 F1 fans
 data science students
 sports analysts (basic level)
 Value Proposition
“Turn complex F1 telemetry into understandable strategy insights and predictions.”
